# haitian-recipe-app

## Description
The app is designed to be used by users who want to save their food recipes

## The database contains three models 
1- Recipes
2- List of items
3- Categories

## Tech used:
- Express js
- node js
- mongodb
- mongoose
- css 
- html
- javascript
- heroku
